# HobbyDIYnSkill

> The human-safe skills marketplace. Learn, create, sell, and master physical crafts that AI cannot replace.

---

## 📋 项目简介

HobbyDIYnSkill 是一个面向手工艺爱好者的一站式平台，提供工具套件、社区作品展示和专业课程培训服务。平台融合 AI 技术，帮助用户学习新技能、展示创意作品并实现变现。

---

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 无需后端依赖，纯前端静态页面

### 运行方式

```bash
# 克隆或下载项目
git clone <repository-url>
cd HobbyHub

# 方式1: 使用浏览器直接打开
open hobbydiy.html

# 方式2: 使用本地服务器（推荐）
python -m http.server 8000
# 然后访问 http://localhost:8000/hobbydiy.html
```

## 🎯 使用示例

### ✅ 已实现功能

**1. 探索调酒配方**
- 点击导航栏的「🍸 Mixology」或直接访问 `mixology.html`
- 使用筛选标签（经典、热带、烈酒、起泡酒、现代）过滤配方
- **长按卡片**触发 Hold-to-Reveal 效果，查看模拟的技能学习历程

**2. 礼物推荐系统**
- 点击顶部导航栏的礼物图标 🎁（带红点标记）
- 查看即将到来的节日倒计时和推荐礼品
- 添加/删除生日提醒（数据持久化到 LocalStorage）

**3. 页面导航**
- 点击分类标签切换不同页面（仅首页和调酒页面有实际内容）
- 点击「Back to Home」返回首页

---

### 🟡 仅展示功能（无交互）

**浏览工具套件**
- 打开 `hobbydiy.html`
- 查看三大板块：工具套件、社区作品、课程训练营
- 卡片仅展示静态数据，点击无详情页

**筛选器**
- 年龄/性别筛选器 UI 存在，但点击无实际筛选效果
- 搜索框存在，但输入无搜索功能

**用户操作**
- 登录/注册按钮仅展示，无功能
- 收藏/购物车入口仅展示，无功能
